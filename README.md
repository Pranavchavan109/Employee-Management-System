# Employee-Management-System
Python-based Employee Management System implementing CRUD operations, OOP principles, and CSV file handling.


##  Project Overview

The Employee Management System is a console-based Python application designed to manage employee records efficiently. It allows users to perform core operations such as adding, viewing, updating, searching, and deleting employee details.

This project demonstrates strong Python fundamentals, Object-Oriented Programming (OOP), file handling, and real-world CRUD operations.



##  Project Objectives

- Build a Python-based employee record management system  
- Implement CRUD operations using clean and modular code  
- Store and retrieve employee data using CSV files  
- Apply OOP principles for better scalability  
- Handle user input validation and exceptions  



##  Features

- Add new employee records  
- View all employees  
- Search employee by ID  
- Update employee details  
- Delete employee records  
- Persistent storage using CSV file  
- Menu-driven user interface  



##  Technologies Used

- Python  
- CSV File Handling  



##  Python Concepts Implemented

- Object-Oriented Programming (Classes & Objects)  
- File Handling (CSV)  
- Functions and Modular Programming  
- Exception Handling  
- Conditional Statements and Loops
